# TRACEFLOW Backend

Backend do TRACEFLOW desenvolvido com Node.js, Express, Prisma e MySQL.

## Estrutura

```txt
src/
|-- config/
|-- database/
|-- modules/
|-- routes/
|-- app.js
`-- server.js
```

## Arquitetura

```txt
Routes -> Controller -> Service -> Repository -> Database
```

## Observacao

Este backend ainda nao implementa regras de negocio. A estrutura esta preparada para desenvolvimento futuro.
